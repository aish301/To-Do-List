To-Do List Application

A simple and user-friendly To-Do List web application built using HTML, CSS, and JavaScript.
This app helps users manage daily tasks efficiently with a clean UI and smooth user experience.

Features
1. Add new tasks
2. Delete tasks
3. Prevent duplicate tasks
4. Automatically trim extra spaces
5. Persist tasks using LocalStorage
6. Tasks remain after page refresh
7. Custom toast notifications for add & delete actions
8. Clean and responsive UI

Tech Stack
HTML – Structure
CSS – Styling & layout
JavaScript – Functionality & logic
